== 管理(Admin)ペイン

admin ユーザの場合、情報コンテキストメニューに `Admin` を表示する。

`Admin` は次を表示する。

* Summary
* Validation
* current.page record
* graph record
* group record
* SVG element
* input target

== ログインユーザメニュー

ノートキャンパス右上のログインアイコンをクリックすると、ログインユーザメニューを開く。

未ログイン時は `Login` のみ表示する。

ログイン後は次を表示する。

* Login user information
** login id
** ユーザ名
* Logout
