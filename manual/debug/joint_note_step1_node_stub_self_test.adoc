== 目的

この手順は、ブラウザを使わず Node 上で `wuwei.debug.runJointStep1NoServerTest()` を実行し、Joint Note Step 1 の import / team / own 判定がすべて `ok: true` になることを確認するためのものである。

Apache24、CGI、共同編集サーバー API は使用しない。

== 前提

作業ディレクトリは `wu_wei2` ルートとする。

[source,powershell]
----
cd C:\Apache24\htdocs\wu_wei2
node --version
----

`index.html` の読み込み順に近い形で、次の 2 ファイルを Node 上で評価する。

* `app2/model/wuwei.joint.js`
* `app2/misc/wuwei.debug.js`

== 実行コマンド

PowerShell で次を実行する。

[source,powershell]
----

global.window = global;

== 期待結果

`console.table` に import / team / own のチェック結果が表示される。

すべての行で `ok` が `true` になり、最後に次が表示されれば成功である。

[source,text]
----
debug self-test ok
----

特に確認する項目は次である。

* import state: `jointNoteState == "imported"`
* import state: `origin.type == "import"`
* import state: `origin.source == "export-package"`
* import state: 他者ノードの soft delete が可
* team state: `jointNoteState == "team"`
* team state: `origin.type == "team"`
* team state: `origin.source == "team-note"`
* team state: 他者ノードの編集及び delete が不可
* own state: `wuwei.joint.getNoteState()` が `own`

== 失敗時の見方

失敗した場合は、`result.checks.filter((item) => !item.ok)` の JSON が表示される。

`actual` と `expected` を比較し、次の観点で切り分ける。

* `wuwei.joint.isImportedNote()` が `origin.type/source` を見ているか。
* `wuwei.joint.isTeamNote()` が `jointNoteState`, `note_scope`, `origin.type/source` を見ているか。
* `wuwei.debug` の `setJointImported()` / `setJointTeam()` が `jointNoteState`, `note_scope`, `origin` を設定しているか。
* `wuwei.note.exportNoteText()` 相当の経路で `origin` が落ちていないか。

== 注意

このテストは Node 上の最小スタブであり、実ブラウザ上の描画、クリック操作、編集ペイン、Apache24 配信、CGI 保存は確認しない。

実画面での確認は `manual/joint_note/joint_note_step1_no_server_test.adoc` の Chrome Console 手順で行う。
