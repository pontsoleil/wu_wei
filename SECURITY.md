# Security Policy

## Supported Versions

WuWei is currently maintained on the `master` branch.

| Version / Branch | Supported |
| --- | --- |
| master | Yes |
| older snapshots | No |

## Reporting a Vulnerability

Please do not disclose security issues publicly before they are reviewed.

Report vulnerabilities by opening a private security advisory on GitHub, or contact the maintainer directly if private advisory access is not available.

When reporting, include:

- Affected URL, script, or file
- Steps to reproduce
- Expected and actual behavior
- Possible impact
- Environment details, if relevant

We will review reports as soon as practical and publish fixes through the repository.
